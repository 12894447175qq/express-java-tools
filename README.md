### 快递api java封装

> 封装京东物流、顺丰物流api

**主要实现了以下功能：**

【1】快递下单

【2】取消下单

【3】快递轨迹查询

---------------------------------

### Maven 引用方式

```xml
<dependency>
  <groupId>club.mrxiao</groupId>
  <artifactId>（不同模块参考下文）</artifactId>
  <version>1.0.0</version>
</dependency>
```

- 京东物流：`express-java-jdl`   
- 顺丰物流：`express-java-sf`
